# Smart Advertising
## Developed on environment
Node: v12.16.3
NPM: v6.14.4
webpack: v4.42.1
Ubuntu: v20.04
Chrome: v81.0.4044.129 (Official Build) (64-bit)

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run start
```

To create a development build:

```sh
npm run build
```
